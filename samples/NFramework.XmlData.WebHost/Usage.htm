XML Data Service 사용법

Methods Description
IHttpHandler
URI /XmlDataService.ashx?Product=[ProductName]&UseCompression=[0|1]
Parameters Product : 사용할 Database Name (web.config의 connectionStrings의 name)
UseCompression : 압축 사용 여부
예제 http://localhost/XmlDataService/XmlDataService.ashx?Product=Northwind&UseCompression=1 : (Northwind DB, 압축 사용)
ASP.NET Pages
URI /XmlDataService.aspx?Product=[ProductName]&UseCompression=[0|1]
Parameters Product : 사용할 Database Name (web.config의 connectionStrings의 name)
UseCompression : 압축 사용 여부
예제 http://localhost/XmlDataService/XmlDataService.aspx?Product=Northwind&UseCompression=1 : (Northwind DB, 압축 사용)
ASP.NET WebService
URI /XmlDataService.asmx
Methods 
byte[] Execute(byte[] requestData, string productName, bool useCompression);
System.Xml.XmlDocument ExecuteToXmlDocument(System.Xml.XmlDocument requestXml, string productName, bool useCompression);
예제 
byte[] responseData = _xdWebService.Execute(requestData, "", useCompression);
XmlDocument responseXml = _xdWebService.ExecuteToXmlDocument(requestXml, "", false);
Method Methods.aspx?Product=ProductName (Method 방식의 Request에 제공되는 Method 정보)
